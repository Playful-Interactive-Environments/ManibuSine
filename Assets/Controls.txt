GameChangerSuite Controls:

GameSelection:
ESC, Q -> Load Credits
F1, 1 -> Load Game "Fluridus 293"
F2, 2 -> Load Game "Tower Of Power"
F3, 3 -> Load Game "Swarm Defender"
F4, 4 -> Load Game "Fish Feast"
F5, 5 -> Load Game "Beelzeball"
P -> toggle pause
R -> reconnect Tracking Service
N -> reload GameSelection

InfoScreen:
Tab -> press and hold to display information about: resolution, tracklink and tuio state

EvaluationTool:
T -> toggle recording of evaluation data (data gets stored in C:\Users\[USER]\AppData\LocalLow\gLab Hagenberg\GameChangerSuite\recording)

Credits:
ESC, Q -> Quit Application (return to Windows)


Config File:

tracklink-on
0 -> Tracklink OFF
1 -> Tracklink ON

tuio-on
0 -> Tuio OFF
1 -> Tuio ON

targetResolution
0 -> 3840 x 1920
1 -> 1920 x 1080
2 -> 1280 x 720
